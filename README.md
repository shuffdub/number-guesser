# Number Guesser

A JavaFX number guessing game. Guess the randomly selected number between 1 and 10.

## Built With
- **Java** — core game logic
- **JavaFX** — graphical user interface
- **Maven** — build and dependency management
- **CSS** — custom styling


## How To Play
- Launch the app and enter your guess
- Get instant feedback if you're right or wrong
- Reset anytime to play again
- Exit the game when you're finished
